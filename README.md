# ДЗ №2 Базы данных

## Основные команды

- Создание пользователя: create_user name:[имя] email:[почта] age:[возраст]
- Обновление данных о пользователе: update_user id:[id] name:[имя] email:[почта] age:[возраст] (обязателен только id)
- Удаление пользователя: delete_user [id]
- Получение данных о пользователе по id: get_user [id]
